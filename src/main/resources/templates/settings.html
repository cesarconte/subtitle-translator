Glossary Usage Guide

SubTranslator supports DeepL glossaries to help you control the translation of specific terms, names, or phrases. This is especially useful for:

  • Ensuring names (e.g., Susan) are not translated.
  • Maintaining consistent translations for technical terms or brand names.
  • Customizing translations for your project or organization.

How to Use a Glossary

  1. Create a glossary in DeepL:
    • Log in to your DeepL Pro account.
    • Go to the Glossaries section and create a new glossary for your language pair.
    • Add the terms you want to control (e.g., Susan → Susan, CPU → CPU).
  2. Select your glossary in SubTranslator:
    • When translating, open Advanced translation options.
    • If you have glossaries, you will see a dropdown to select one. If not, you can enter a glossary ID manually.
  3. Translate as usual: The selected glossary will be applied, and your specified terms will be preserved or translated as you defined.

Troubleshooting

  • If no glossaries appear, make sure you have created at least one in your DeepL account for the selected language pair.
  • Glossaries are only available for certain language pairs supported by DeepL.
  • If you enter an invalid glossary ID, the translation may fail or ignore the glossary.

For more details, see the DeepL API Glossaries documentation.
